# AI-DIET-PLAN

A smart diet planner web app built with React + Vite and styled with Tailwind CSS.

## Features
- Personalized diet plan generator with calorie & macro calculator
- Today's mindful meal menu (Breakfast, Lunch, Dinner)
- Diet tips bento grid
- Success stories / testimonials
- Email newsletter CTA
- Fully responsive design

## Tech Stack
- React 19
- Vite 8
- Tailwind CSS v4
- Groq AI API

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the root:

```
VITE_API_KEY=your_groq_api_key
```
